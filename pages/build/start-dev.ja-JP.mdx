# 開発の始め方

1. `pnpm install`

2. `pnpm build`と`pnpm dev`の２つの選択肢があります。
`pnpm build`は配布用のビルドで、`pnpm dev`は自動リロード機能のあるデバッグ用です。
`pnpm dev`の使い方は「デバッグの始め方」をご参照ください。

# デバッグの始め方

## Windows

1. zstdをインストールし、PATHに追加します。
2. [noraneko-runtime Action](https://github.com/nyanrus/noraneko-runtime/actions/workflows/wrapper_windows_build.yml)で最後に成功したビルドアーティファクトをダウンロードします。
3. そのzipファイルを解凍し、nora-win_x64-bin.tar.zstファイルをプロジェクトルートへ置きます。
4. `pnpm dev`を実行します
5. ブラウザが実行されます。ソースコードを変更した場合、リロードまたはブラウザの再起動が自動で行われます。
