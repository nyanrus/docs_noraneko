# Introduction

### Welcome to the Noraneko Docs!

Noraneko is the base of Floorp 12, built using a modern JS/TS environment.

### Additional Resources

Noraneko was originally designed for Floorp, so the Floorp Community can assist you. If you’re interested in Noraneko’s development status, please visit the Ablaze Community Discord!
